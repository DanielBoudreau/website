JMRI NMRA 2008 Clinics

This page contains links to clinics that were presented at the NMRA 2008 convention in Anaheim. In addition, our traditional annual JMRI user meeting was held during the convention.

David Duchamp's clinics

  • The Intro to Decoder Programming with JMRI/DecoderPro clinic PowerPoint is available as a PDF file (2.2MB).
  • The Using the JMRI/PanelPro Layout Editor clinic is available as a PDF file (7MB).
  • The LayoutEditorClinic.xml and LayoutEditorClinicCTC.xml panel files are also available.

Dick Bronson's clinics

  • The Introduction to PanelPro clinic PowerPoint slides and XML layout configuration files are available from Dick's own website. He also has a number of other interesting and useful JMRI clinics there as well.

Several of the clinics were filmed by Model Railroad Hobbyist magazine, and are available from their website.
